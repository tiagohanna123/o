# PROBLEMA 1: MATÉRIA ESCURA E ENERGIA ESCURA

## SOLUÇÃO DERIVADA

### Massa da Partícula de Matéria Escura

**Valor calculado:** `m_DM = 7.32 × 10⁵ eV/c² ≈ 732 keV/c²`

**Derivação:**
```
m_DM = k_B × T_desacoplamento × (Ω_DM/Ω_DE)^(1/3) / c²

Onde:
- T_desacoplamento ≈ 1.16 × 10¹⁰ K (escala MeV)
- Ω_DM/Ω_DE = 0.392 (razão observada)
```

**Interpretação:** A matéria escura é composta de WIMPs leves ou axions massivos na faixa de centenas de keV.

### Escala de Energia da Energia Escura

**Valor calculado:** `E_DE = 2.24 meV`

**Derivação:**
```
ρ_DE = (E_DE)⁴ / (ℏc)³
E_DE = (ρ_DE × ℏ³c³)^(1/4) = 2.24 meV
```

# PROBLEMA 2: GRAVIDADE QUÂNTICA

## SOLUÇÃO DERIVADA

### Constante de Acoplamento Quântico-Gravitacional

**Valor calculado:** `κ_MX = 3.36 × 10⁻⁷⁸ m`

**Derivação:**
```
G_μν = κ_MX × ∇(S/E) × T_μν

κ_MX = κ_GR / ∇(S/E)_Planck
κ_MX = (8πG/c⁴) / (1/l_P)
κ_MX = 8πG × l_P / c⁴
```

### Energia de Unificação

**Valor calculado:** `E_unif = 7.90 × 10¹⁸ GeV`

**Derivação:**
```
E_unif = E_Planck × (α × β / γ)^(1/4)
E_unif = 1.22 × 10¹⁹ × (0.3 × 0.7 / 1.2)^(1/4)
E_unif = 7.90 × 10¹⁸ GeV
```

### Resolução do Problema dos Infinitos

**Método:** Limite natural `S + E ≤ 1`

```
Não há divergências porque:
1. S e E são limitados entre 0 e 1
2. A soma S + E ≤ 1 é condição de conservação
3. Cutoff natural em E_Planck emerge automaticamente
```

# PROBLEMA 3: ORIGEM DA VIDA

## SOLUÇÃO DERIVADA

### Energia Crítica de Transição

**Valor calculado:** `E_crítico = 28.0 kJ/mol`

**Derivação:**
```
E_crítico = E_ATP × (S_vida - S_prebiótico) / (1 - E_vida)
E_crítico = 30.5 × (0.60 - 0.05) / (1 - 0.40)
E_crítico = 30.5 × 0.55 / 0.60 = 28.0 kJ/mol
```

### Temperatura Crítica

**Valor calculado:** `T_crítico = 1353 K (1080°C)`

**Derivação:**
```
T_crítico = E_crítico / [R × ln(S_vida/S_prebiótico)]
T_crítico = 28000 / [8.314 × ln(0.60/0.05)]
T_crítico = 28000 / [8.314 × 2.48] = 1353 K
```

### Probabilidade de Abiogênese

**Valor calculado:** `P(vida) = 1.0` (praticamente certa dado tempo suficiente)

**Condições necessárias:**
- Energia disponível > 28 kJ/mol
- Temperatura: 320-380 K (fontes hidrotermais)
- Concentração orgânica > 0.95 mM
- Água líquida presente
- Tempo > 10⁸ anos

### Dinâmica de Resistência

**Tempo para dominância:** `~5 horas` sob pressão antibiótica

**Equação:**
```
df_R/dt = f_R × (1-f_R) × (μ_R - μ_S × (1-kill))

De f_R = 1% para f_R = 90%:
t = ln(90/1) / μ_R ≈ 5.6 horas
```

### Estratégia Ótima

1. **Reduzir uso em 50%** → reversão em 5 anos
2. **Combinação de 3 antibióticos** → P(resistência) < 10⁻¹⁶
3. **Ciclos a cada 3 meses** → mantém custo fitness
4. **Dose alta e curta** → mata antes de resistência
